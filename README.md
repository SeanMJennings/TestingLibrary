# testing

Dotnet packages to aid testing in a behavioural-driven development (BDD) style.

```
    [Test]
    public void pass_our_first_behavioural_test()
    {
        Given(two_numbers);
        When(we_give_them_to_our_complex_system);
        Then(we_get_the_sum);
        And(we_can_validate_something_else);
    }
```
